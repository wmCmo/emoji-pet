# Emoji Pet

- This is a fun little project I made when I was bored.
- You can interact with him by hovering or clicking.
- When it’s nighttime, he’ll go to sleep.
- Sometimes he gets hungry or bored, so don’t forget to feed and play with him.
- Suggestions or contributions are always welcome!
